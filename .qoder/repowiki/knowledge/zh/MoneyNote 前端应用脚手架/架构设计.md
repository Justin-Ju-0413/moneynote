- 采用 Vite 作为构建工具，集成 `@vitejs/plugin-react` 进行 React 支持。
- 使用 Tailwind CSS v4 及其 Vite 插件进行样式管理。
- 配置 `vite-plugin-pwa` 实现渐进式 Web 应用（PWA）功能，包含自动更新和离线支持。
- 通过 `path.resolve` 设置 `@` 别名指向 `./src` 目录，规范模块导入路径。
- 采用 TypeScript 项目引用（Project References）模式，将配置拆分为 `tsconfig.app.json` 和 `tsconfig.node.json`。
- 使用 `typescript-eslint` 和 `eslint-plugin-react-hooks` 建立代码质量与规范检查体系。
